Bootstrap 5 Z-index 层级控制

1. 预定义层级

Bootstrap 5 内置了以下层级等级：

z-3 (最高)
z-2
z-1
z-0 (最低)
层级值越大，元素显示在越上层

2. 组件默认层级

Bootstrap 组件有各自的默认层级值：

    $zindex-dropdown:        1000;
    $zindex-sticky:          1020;
    $zindex-fixed:           1030;
    $zindex-offcanvas:       1040;
    $zindex-modal-backdrop:  1050;
    $zindex-modal:           1060;
    $zindex-popover:         1070;
    $zindex-tooltip:         1080;

3. 自定义层级

HTML 使用方式
<div class="z-index-9999">自定义层级元素</div>
Sass 自定义方式
    $custom-zindex: 9999;
    .custom-element {
      z-index: $custom-zindex;
    }

Bootstrap 5 最佳实践

工具类使用原则

优先使用工具类

减少自定义CSS代码量，提高开发效率

<div class="p-3 m-2 bg-light rounded shadow-sm">内容</div>
响应式设计

结合断点使用，实现不同屏幕尺寸下的最佳展示效果

<div class="mt-2 mt-md-4 p-3 p-lg-5">响应式内容</div>

层级管理原则

普通元素

使用 z-0 到 z-3 的层级

  • z-0：最底层
  • z-1：中间层
  • z-2：较高层
  • z-3：普通元素最高层
组件层级

使用预定义层级值

  • 下拉菜单: 1000
  • 粘性元素: 1020
  • 固定元素: 1030
  • 模态框: 1060
避免的做法
避免使用随机大值

不推荐: z-index: 9999;

不推荐: z-index: 100000;

这会导致层级冲突和管理混乱

绝对定位

绝对定位元素需要父级设置 position-relative
