Bootstrap 5 Layout Grid

Grid Introduction

Bootstrap 5 Layout Grid system 是响应式的，配备了 flexbox（弹性盒子），并允许页面最多分为 12 列。您可以将列组合在一起。这个网格系统是一个强大的移动优先响应式 flexbox（弹性盒子）网格。

其工作原理如下：

  • 网格系统支持六个响应式断点。您可以通过每个断点来控制 container（容器）、column（列）的大小和行为。
  • Containers（容器）用于居中并水平填充您的内容。使用 .container 类来实现响应式宽度，使用 .container-fluid 类来实现所有 viewport（视口）和 device（设备）上的 100% 宽度。使用 .container-md 类来同时实现 fluid（流体）宽度和 pixel（像素）宽度。
  • Rows（行）是 columns（列）的 wrapper（包装器）。每列都有水平 padding（内边距，称为 gutter（沟槽）），用于控制它们之间的空间。
  • Columns（列）非常灵活。每行有 12 个 template columns（模板列）可供使用，允许您创建跨越任意数量列的不同元素组合。
  • Gutters（沟槽）也是响应式的，并且可以在所有 breakpoints（断点）上进行自定义。我们可以使用 .gx-* 来改变 horizontal gutters（水平沟槽），使用 .gy-* 来改变 vertical gutters（垂直沟槽）。

Bootstrap 5 Layout Grid System 的特点：

  • Grid options（网格选项）：Bootstrap 的网格系统适用于所有六个 breakpoints（断点），即 xs、sm、md、lg、xl、xxl。
  • Auto-layout columns（自动布局列）：这提供了无需显式 numbered class（编号类）即可轻松创建列大小的功能。
  • Responsive classes（响应式类）：它们提供了 predefined classes（预定义的类），用于构建复杂的响应式、动态和灵活的布局。
  • Nesting（嵌套）：我们需要为 nesting（嵌套）添加一个新的 row（行）和一组 columns（列），这包括一组加起来等于或少于 12 的列。
  • Sass：该系统还提供了 Sass variables（变量）和 mixins（混合），用于创建自定义和响应式的页面布局。
  • Customizing the grid（自定义网格）：这用于通过使用内置的 grid Sass variables（网格 Sass 变量）和 map（映射）来按需自定义您的网格。

Bootstrap 5 Layout Column

Bootstrap 5 列（Columns）通过使用 flexbox 网格系统，提供了对齐、排序和偏移列的不同选项。可以利用不同的列类来管理非网格元素的宽度。

Bootstrap 5 列：以下组件用于帮助垂直和水平对齐列，并重新排序它们，具体描述如下：

对齐（Alignment）

它利用 flexbox 对齐来垂直或水平排列项目。它有四种对齐方式：

  • 垂直对齐（Vertical alignment）：用于垂直对齐项目。
  • 水平对齐（Horizontal alignment）：用于水平对齐项目。
  • 列换行（Column Wrapping）：用于将溢出的列换行到新的一行。
  • 列断行（Column Breaks）：用于将列断行到新的一行。

重新排序（Reordering）

它用于根据 HTML 元素上给定的 Bootstrap 5 类重新排列列中的项目。它有两种重新排序方式：

  • 排序类（Order Classes）：用于使用 order 类控制内容的视觉顺序，这些类是响应式的。
  • 偏移列（Offsetting Columns）：用于使用 offset 网格类控制内容的顺序。

独立列情况（Standalone Column Cases）

默认情况下，它用于为元素提供特定宽度，且没有内边距（padding）。

Bootstrap 5 Layout Gutter

在 Bootstrap 的网格布局中，Gutters（沟槽）指的是列之间的间隙。可以通过使用 gutter 修改类来改变这些沟槽的宽度。

Bootstrap 5 布局 Gutter 修改类：

  • g-*：此类用于设置所有方向（水平和垂直）的沟槽宽度。
  • gx-*：此类用于设置水平方向上的沟槽宽度。
  • gy-*：此类用于设置垂直方向上的沟槽宽度。

其中 * 可以取值为 0 到 5。数值越大，间隙越宽。
